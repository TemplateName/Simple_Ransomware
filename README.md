<h1>Simple Ransomware</h1>

 ### [Project Inspiration](https://www.youtube.com/watch?v=UtMMjXOlRQc)

<h2>Description</h2>
Project consists of creating a simple Python script that acts like ransomware, altering the contents of every files in a directory and demanding the user a ransom to restore the files back. The project also reverts the files back to its original state.
<br />


<h2>Languages and Utilities Used</h2>
 
- <b>Python</b>

<h2>Environments Used </h2>

- <b>Linux box (Linode server)</b> (Ubuntu 22.04 LTS)
